ユーザインターフェースデザイン、ソフトウェア開発

美しく直感的なウェブサイト、ソフトウェアを作ります


得意領域：

  • ユーザエクスペリエンスデザイン、ユーザインターフェースデザイン
  • プロトタイピング
  • フロントエンドソフトウェア開発

経験：

  • フロントエンド／バックエンドソフトウェア開発
  • レスポンシブ　デザイン(携帯端末への最適化)
  • システム開発全工程（要件定義-設計-実装-テスト-導入）
  • ウェブシステム、Microsoft Windows デスクトップアプリケーション、携帯アプリケーション
  • MVC フレームワーク、 ビジネスインテリジェンス、UML
  • データ設計、 パフォーマンス最適化
  • プログラム言語: C# (.NET Framework)、 VB Script (ASP)、 Java Script (JQuery、 Backbone、 Node.js、 JSON)、 Lisp、 SQL
  • マークアップ言語: HTML/HTML5、 CSS/CSS3、 XML Parsing and XSLT
  • DB: Microsoft SQL Server、 Oracle Database
  • ソフトウェア開発モデル: ウォーターフォール、 アジャイル
  • 使用ソフトウェア: Adobe Creative Suites、 Microsoft Office Suites、 バージョン管理: SVN、 Git
